直接把刚才博客的命令搬过来，都是常用的：
1. 由用户视图进入系统视图

```
system-view
```
2. 由用户视图进入系统视图

```
sysname R1
```
3. 查看路由器当前配置信息
```
display current-configuration
```
4. 查看接口信息
```
display interface
```
5. 进入接口视图
```
interface type number
```
注意，交换机必须进入系统视图才能去配置，因为系统视图的权限比较大，可作参数修改，设定系统功能，比如划分VLAN，配置路由，设定访问方法，设定速率；用户视图权限小，只能看看基本配置信息，只读的，不能修改。
## 三、实验步骤
摸索差不多了，开始继续实验内容，根据这篇[博客](https://blog.51cto.com/jackyzhang/1034893)的内容，我们要对交换机操作一下：
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/46243b8242aac7d880986f5458983fad.png)
这里先启动，然后打开CLI（个人理解就是交换机的命令行），然后输入代码：

```
system-view
sysname R1
vlan batch 10 20
```
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/283386e838a0c1fe4f7c554a7d72b3f5.png)
可以看到方括号[]是系统视图，<>尖括号是用户视图。然后继续定义接口：

```
interface GigabitEthernet 0/0/1
port link-type access
port default vlan 10
quit
```
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/f8ce4816b418843718fdbe7b1fc1cbcc.png)这样第一个端口就设置好了。接下来设置第二个端口
```
interface GigabitEthernet 0/0/2
port link-type access
port default vlan 20
quit
```
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/fba373a3b4c8a76c3c8f1f6748020dc0.png)
然后要让两个交换机连接，所以要设置trunk接口

```
interface GigabitEthernet 0/0/24
port link-type trunk
port trunk allow-pass vlan 10 20
quit
```

然后是下一台交换机，注意这些参数不是乱写的，是照着题目要求。
![在这里插入图片描述](https://i-blog.csdnimg.cn/blog_migrate/6d8b45e010b5dc0290ac2bd7087b7be0.png)
```
system-view
sysname R2
vlan batch 10 20
```
```
interface GigabitEthernet 0/0/1
port link-type access
port default vlan 10
quit
```
```
interface GigabitEthernet 0/0/2
port link-type access
port default vlan 20
quit
```
```
interface GigabitEthernet 0/0/24
port link-type trunk
port trunk allow-pass vlan 10 20
quit
```
