# 一、安装

这个软件是跑命令行然后在web界面操作，既然是这样的话，就不在本地下载了，直接在服务器上搞

记得要：
```bash
chmod +x ./openresearch-linux-x64
# 启动
./openresearch-linux-x64
```

哈哈，作者服务端口卡死127.0.0.1，还不能内网访问，那只能在本地用了... 吗?

我不想在本地用，我目测这个东西是个大号obsidian， 自然是要架在服务器上的，自己搞个端口转发好了
```bash
sudo nano /etc/systemd/system/openresearch.service
```

```bash
[Unit]
Description=OpenResearch App Service
After=network.target

[Service]
# 指向你软件所在的目录
WorkingDirectory=/opt/openresearch
# 执行命令，确保有执行权限 (chmod +x openresearch-linux-x64)
ExecStart=/opt/openresearch/openresearch-linux-x64
Restart=always
User=user

[Install]
WantedBy=multi-user.target
```

然后写一个端口转发一下，首先安装：
```bash
sudo apt install socat
```
然后端口转发
```bash
sudo nano /etc/systemd/system/port-forward.service
```

```
[Unit]
Description=Port Forwarding Service
# 重点：确保在软件启动后才开始转发
After=openresearch.service
Requires=openresearch.service

[Service]
ExecStart=/usr/bin/socat TCP4-LISTEN:4096,fork,reuseaddr TCP4:127.0.0.1:40405
Restart=always
User=user

[Install]
WantedBy=multi-user.target
```

最后启动：
```bash
sudo systemctl daemon-reload
sudo systemctl start openresearch
sudo systemctl start port-forward
```
