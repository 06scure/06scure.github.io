## 1. 报错信息

OSError You seem to have cloned a repository without having git-lfs installed. Please install git-lfs and run git lfs install followed by git lfs pull in the folder you cloned.

操作系统错误：您似乎在没有安装git lfs的情况下克隆了一个存储库。 
请在克隆的文件夹中安装git lfs并运行“git lfs install”，然后运行“git lf pull”。
## 2. 解决方案

git clone预训练模型的时候clone下来的只是一个指针，难怪秒下完，还是要用git lfs下载...

使用的是服务器环境，因此可以用conda git-lfs部署：
```bash
conda install -c conda-forge git-lfs
```
安装好之后，使用命令查看版本：
```bash
git lfs --version
```
```
git-lfs/3.7.0 (GitHub; linux amd64; go 1.24.4)
```
git lfs初始化：
```bash
git lfs install
```
```
Git LFS initialized.
```
之后进入模型的文件夹目录，运行：
```bash
git lfs pull
```
即可下载模型，问题解决。
