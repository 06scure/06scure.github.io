pycharm只适合调试代码，长时间训练应当使用tmux配合进行。
### 1. 启动与退出
```bash
# 启动
tmux
# 退出
exit
```
### 2. 会话管理
```bash
# 新建一个会话
tmux new -s <name>
# 分离会话
tmux detach
# 查看会话
tmux ls
# 接入会话
tmux attach -t <name>
# 杀死会话
tmux kill-session -t <name>
# 切换会话
tmux switch -t <name>
# 重命名会话
$ tmux rename-session -t 0 <name>
```
